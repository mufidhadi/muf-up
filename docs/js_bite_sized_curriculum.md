# Kurikulum Bite-Sized JS & TS (Edisi 2025)

Daftar materi mikro (micro-learning) untuk menguasai JavaScript dan TypeScript secara bertahap dalam waktu 15-30 menit per topik.

---

## 🟢 Modul 1: Fondasi Modern
1.  **The Engine:** Memahami Runtime JS (Node, Bun, Deno) dan eksekusi kode.
2.  **Strict Mode:** Menggunakan `"use strict"` dan kelebihan ES Modules (`import`).
3.  **Variable Scope:** Perbedaan `const`, `let`, dan cara kerja Temporal Dead Zone.
4.  **String Magic:** Template Literals dan manipulasi string modern.
5.  **Arrow Power:** Menulis fungsi ringkas dengan Arrow Functions.
6.  **De-structuring:** Teknik membongkar Array dan Object dengan cepat.
7.  **Spread & Rest:** Penggunaan `...` untuk manipulasi data tanpa mutasi.
8.  **The Waitlist:** Memahami `async / await` dan `Promise`.
9.  **Date Overhaul:** Menggunakan **Temporal API** (Say goodbye to `Date`).

---

## 🟡 Modul 2: Data & Logic Mastery
10. **Immutable Arrays:** Menggunakan `toSorted`, `toReversed`, dan `toSpliced`.
11. **Smart Sets:** Operasi Set asli (`union`, `intersection`, `difference`).
12. **The Resolver:** Menggunakan `Promise.withResolvers()` untuk logika kompleks.
13. **Modern Classes:** Private fields (`#`) dan static methods.
14. **Iterator Helpers:** Menggunakan `.map()` dan `.filter()` pada iterators.
15. **Pattern Matching:** Alternatif bersih untuk `switch` (ES2025 proposal).
16. **Import Attributes:** Cara aman mengimpor JSON dan CSS di JS.

---

## 🟠 Modul 3: TypeScript Foundations
17. **Why Types?** Setup TypeScript dan memahami kelebihannya.
18. **Basic Types:** String, Number, Boolean, Array, dan Null/Undefined.
19. **Interface vs Type:** Kapan menggunakan masing-masing?
20. **Union & Intersection:** Menggabungkan tipe data secara fleksibel.
21. **Generic Functions:** Menulis fungsi yang mendukung banyak tipe data.
22. **Utility Types:** Mengenal `Partial`, `Pick`, `Omit`, dan `Readonly`.
23. **Runtime Safety:** Validasi schema data input dengan **Zod**.

---

## 🔴 Modul 4: Frontend Frameworks (React 19)
24. **The Starter:** Setup project React 19 dengan Vite.
25. **JSX Mastery:** Menulis komponen reaktif yang efisien.
26. **Server Components:** Memahami pemisahan client vs server side.
27. **Action Hooks:** Penggunaan `useActionState` untuk form handling.
28. **State with Zustand:** Management state global tanpa boilerplate.
29. **Signals:** Konsep reaktivitas alternatif untuk performa tinggi.
30. **Tailwind v4:** Styling cepat dengan engine CSS baru.

---

## 🟣 Modul 5: Backend & APIs
31. **Fast Runtimes:** Menjalankan script dengan **Bun** (Startup < 50ms).
32. **The Edge Server:** Membuat API dengan **Hono**.
33. **Postgres Lab:** Setup database PostgreSQL dan koneksi ke Node/Bun.
34. **The ORM:** Menggunakan **Drizzle ORM** untuk query type-safe.
35. **Type-safe API:** Komunikasi Frontend-Backend dengan **tRPC**.
36. **Testing with Vitest:** Menulis Unit Test secepat kilat.
37. **E2E with Playwright:** Simulasi interaksi user di browser.

---

## 🔵 Modul 6: AI & Architecture
38. **The Monorepo:** Setup **Turborepo** untuk manajemen multi-app.
39. **AI SDK:** Integrasi LLM (OpenAI/Gemini) ke aplikasi JS.
40. **MCP Protocol:** Membuat server MCP sederhana untuk AI Agents.
41. **Cloud Native:** Deploy aplikasi ke Vercel atau Railway.
42. **CI/CD Pipeline:** Otomatisasi linting dan testing di GitHub Actions.

---
*Daftar ini adalah panduan praktis untuk mengisi materi **muf-up**.*
