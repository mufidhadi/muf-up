# Kurikulum Bite-Sized Python (Edisi 2025)

Daftar materi ini dirancang untuk pembelajaran mikro (micro-learning). Setiap poin adalah satu topik spesifik yang bisa diselesaikan dalam 15-30 menit.

---

## 🟢 Modul 1: Persiapan & Sintaksis Dasar
1.  **The Workshop:** Instalasi `uv` dan setup VS Code (Pylance + Ruff extension).
2.  **Hello 2025:** Menjalankan script pertama dengan `uv run`.
3.  **Memory Box:** Variabel dan tipe data dinamis.
4.  **Talking Types:** String, Integer, Float, dan Boolean.
5.  **Smart Labels:** Type Hinting dasar (`name: str`).
6.  **Decision Point:** Logika `if`, `elif`, dan `else`.
7.  **The Switch:** Menggunakan `match-case` untuk logika yang lebih bersih.
8.  **Looping (Part 1):** Perulangan `for` pada list.
9.  **Looping (Part 2):** Perulangan `while` dan kontrol `break/continue`.

---

## 🟡 Modul 2: Struktur Data & Fungsi
10. **The Collection:** Membuat dan memanipulasi List.
11. **Slicing Mastery:** Teknik memotong List dan String.
12. **The Dictionary:** Menyimpan data dengan Key-Value pairs.
13. **Unique & Fixed:** Mengenal Sets dan Tuples.
14. **One-Line Magic:** Membuat List Comprehensions yang efisien.
15. **Reusable Logic:** Menulis fungsi dengan `def`.
16. **Flexible Arguments:** Menggunakan `*args` dan `**kwargs`.
17. **Anonymous Power:** Mengenal Lambda Functions.
18. **Self-Documenting:** Menulis Docstrings yang profesional (PEP 257).

---

## 🟠 Modul 3: Pemrograman Berorientasi Objek (OOP)
19. **The Blueprint:** Membuat Class dan Instance (Object) pertama.
20. **Self-Reference:** Memahami parameter `self` dalam method.
21. **Modern Data:** Menggunakan `@dataclass` untuk struktur data bersih.
22. **The Family Tree:** Konsep Inheritance (Pewarisan).
23. **Composition:** Menghubungkan antar object tanpa inheritance.
24. **The Safety Net:** Menangani error dengan `try / except`.
25. **Custom Alerts:** Membuat Class Exception sendiri.

---

## 🔴 Modul 4: Ekosistem Modern & Performa
26. **Context Masters:** Menggunakan `with` statement (File Handling).
27. **The Smart Generator:** Menghemat memori dengan `yield`.
28. **Code Enhancers:** Membuat Decorator sederhana.
29. **Instant Cleanup:** Linting dan Formatting otomatis dengan `Ruff`.
30. **Fast Package:** Mengelola library dengan `uv add` dan `uv lock`.
31. **Strict Typing:** Validasi tipe data statis dengan `mypy`.
32. **Secrets Locker:** Mengelola `.env` untuk keamanan API key.

---

## 🟣 Modul 5: Advanced & Concurrency
33. **The Waiting Room:** Konsep dasar `async` dan `await`.
34. **Async Mastery:** Menjalankan banyak tugas sekaligus dengan `asyncio`.
35. **Data Contracts:** Validasi data input dengan **Pydantic Models**.
36. **Magic Methods:** Mengenal Dunder Methods (`__str__`, `__repr__`, `__call__`).
37. **Bulletproof Code:** Menulis Unit Test pertama dengan `pytest`.
38. **The Simulation:** Menggunakan Mocking dalam pengujian.
39. **API Starter:** Membuat satu endpoint dengan **FastAPI**.

---

## 🔵 Modul 6: Spesialisasi AI & Data (2025 Path)
40. **Fast Analytics:** Analisis data kilat dengan **Polars** (Bukan Pandas).
41. **Vector Power:** Operasi array dasar dengan NumPy.
42. **AI Chain:** Pengenalan **LangChain** untuk integrasi LLM.
43. **Data Pipeline:** Dasar-dasar ETL (Extract, Transform, Load).
44. **Dockerize:** Membungkus aplikasi Python ke dalam Container.
45. **Auto-Pilot:** Setup GitHub Actions untuk testing otomatis.

---
*Materi ini dirancang untuk kurikulum **muf-up** agar siswa tidak merasa kewalahan.*
