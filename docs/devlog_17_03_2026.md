## 📊 Task & Troubleshooting Matrix

| Task | Issue / Obstacle | Tried Approach | Result | Lesson Learned |
| :--- | :--- | :--- | :--- | :--- |
| **Riset Kurikulum 2025** | Perubahan standar industri yang cepat (AI, Tooling). | Deep research menggunakan Google Search untuk Python, Go, JS, dan Rust. | Dokumen kurikulum standar & bite-sized di folder `docs`. | Standar 2025 sangat berfokus pada kecepatan (`uv`, `Bun`, `Rust`) dan integrasi AI (`LangChain`, `Candle`). |
| **Update YAML Otomatis** | File YAML sangat besar (8000+ baris) menyebabkan context window penuh dan error `\n` literal. | Menggunakan script Python (`ruamel.yaml`) untuk surgical update materi satu per satu. | Materi Python 45 materi (100% complete) dengan format rapi. | Jangan memanipulasi file besar secara manual; gunakan script untuk menjaga integritas struktur dan encoding. |
| **Frontend Crash (Layar Putih)** | Aplikasi blank setelah update YAML karena data terpotong atau null. | Menambahkan "Null-Safe" logic (Optional Chaining `?.`) di `App.tsx` dan memvalidasi YAML via `uv run`. | Aplikasi kembali berjalan stabil meski data sedang diupdate. | Selalu implementasikan fallback/default value pada inisialisasi state yang bergantung pada data eksternal. |
| **Deployment VPS** | FTP tertutup (ECONNREFUSED) dan SSH memiliki limitasi panjang perintah (Command too long). | Mengalihkan workflow ke **Git Deployment** (GitHub Push & Pull) dan menggunakan **Multi-stage Docker Build**. | Aplikasi live di [muf-up.masmuf.cloud](https://muf-up.masmuf.cloud). | Git adalah cara paling handal untuk sinkronisasi kode antara lokal dan VPS dibandingkan transfer file manual. |
| **Gateway Timeout (Traefik)** | Traefik tidak bisa menjangkau container karena beda network dan port mapping salah. | Menghubungkan Traefik ke network `web`, membersihkan container lama, dan menggunakan Nginx port 80. | Aplikasi dapat diakses via HTTPS secara sempurna. | Traefik harus berada di network yang sama dengan target container agar load balancing berfungsi. |
